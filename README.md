🏥 Doctor Appointment System  

📌 Description  
This is a Spring Boot based backend project for managing doctor appointments.  
It provides REST APIs to perform CRUD operations and manages relationships between Doctor, Patient, and Appointment using MySQL database.  

---

🚀 Features  
➕ Create Doctor, Patient, Appointment  
📄 View All Records  
✏️ Update Appointment  
❌ Delete Appointment  
🔗 Manage relationships (Doctor ↔ Patient via Appointment)  

---

🛠️ Technologies Used  
- Java  
- Spring Boot  
- Spring Data JPA  
- MySQL  
- Postman
